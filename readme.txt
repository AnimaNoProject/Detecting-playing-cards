Um eine Spielsimulation zu starten muss die Main Methode ausgeführt werden.
Sie akzeptiert einen Parameter, welcher ein Pfad zu einem Ordner ist in dem sich beliebig viele Spielzüge (Bilder) befinden.

Beispielspielzüge befinden sich z.B. in input/Datensaetze/Spielsimulation/Spiel 1


Hinweis: Im Spielalgorithmus wird weder Farbzwang, noch das Adut beachtet. Wer zuerst die Karte ausspielt wird aber beachtet 
(Immer der Gewinner der vorherigen Runde. In der ersten Runde beginnt Spieler 1).
